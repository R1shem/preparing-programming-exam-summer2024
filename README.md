# preparing-programming-exam-summer2023
Тренировочный вариант экзамена второго курса.

WPF-приложение, написанное в Visual Studio на C#. Ряд скриншотов Visio-документов 

Функционал: авторизация в системе. Просмотр, запись и чтение информации об аккаунтах и перелётах через текстовые файлы. Вывод данных в Excel.

В целях подготовки к двухчасовому экзамену были выполнены следующие задания:
- написать приложение с использованием текстовых файлов в качестве базы данных для регистрации, авторизации и основных операциях, описанных в предметной области, с последующим выводом в Excel;
- составить диаграммы прецедентов, деятельности, классов, последовательности и ER в соотвветствии с ТЗ.
